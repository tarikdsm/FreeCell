# Roadmap

## Phase 1: Landed In This Delivery

- monorepo bootstrap
- Rust engine with deterministic deals, replay, undo/redo, autoplay, and action masks
- Wasm and Python bindings
- first PixiJS browser client
- Playwright, Python smoke tests, and benchmark scaffolding
- architecture and audit documentation

## Phase 2: Immediate Next Work

- drag-and-drop stack interaction
- hint API backed by ordered legal actions
- stronger regression seed corpus
- richer audio and asset pipeline
- WebKit E2E coverage

## Phase 3: Solver And AI Expansion

- dedicated solver crate or service
- tensor observation encoder
- Gymnasium wrapper
- batched rollout tooling
- seed difficulty labeling and curriculum tooling

## Phase 4: Product Polish

- theme system and alternate card sets
- refined accessibility pass
- saved sessions and statistics
- advanced win and deal animations
- richer telemetry for profiling and UX tuning
